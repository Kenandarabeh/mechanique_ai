# Logo Setup

## Instructions

Please add your logo file as `logo.png` in this directory (`public/logo.png`).

### Recommended specifications:
- **Format**: PNG with transparent background
- **Size**: 512x512px or 1024x1024px (square)
- **Color**: Works with both light and dark modes
- **File name**: Must be exactly `logo.png`

### Current behavior:
- If `logo.png` exists: Your custom logo will be displayed in the sidebar
- If `logo.png` is missing: A default car icon placeholder is shown

The logo will be displayed in the sidebar header with professional styling and animations.
